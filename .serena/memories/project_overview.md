RichZennEditor is a TypeScript web app built with Rsbuild + React 19. It currently contains a minimal TipTap-based editor prototype intended to evolve into a WYSIWYG editor for writing Zenn articles/books. Top-level structure: src/ for app code, src/tiptap/ for editor-related code, tests/ for Rstest + Testing Library tests, config files for Rsbuild, TypeScript, Biome, and Rstest at repo root.
